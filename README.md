# COACHDESK — پلتفرم مربی و ورزشجو 🏋️

پلتفرم ارتباط مربی باشگاه با شاگردان، با ظاهر تیره به سبک Freeletics.
**پشته:** React (Vite) + Supabase + Vercel

## امکانات
- مربی: ساخت حساب ورزشجو، کتابخانه حرکات با ویدیو، برنامهٔ تمرینی هفته/روز، رژیم غذایی، مشاهدهٔ گزارش‌ها
- ورزشجو: تب کوچ با مسیر تمرینی ۱۲ هفته‌ای و کارت‌های جلسه، رژیم، ثبت گزارش، پروفایل با نمودار پیشرفت
- شبکهٔ اجتماعی: فید پست با لایک/کامنت، استوری ۲۴ ساعته، دنبال‌کردن، حریف تمرینی
- موبایل: نوار پایین اینستاگرامی + سایدبار کشویی

## راه‌اندازی محلی
```bash
npm install
cp .env.example .env    # مقادیر Supabase را داخلش بگذار
npm run dev
```

## دیتابیس
فایل `supabase_setup.sql` را در Supabase → SQL Editor اجرا کن (جدول‌ها + RLS + Storage + مربی پیش‌فرض).

## متغیرهای محیطی
| نام | مقدار |
|-----|-------|
| `VITE_SUPABASE_URL` | آدرس پروژهٔ Supabase |
| `VITE_SUPABASE_ANON_KEY` | کلید anon public |
